**Z-Score Threshold chart (heatmap, 280px):**

Pivot the flat results array into a 2D z-matrix for a Plotly heatmap. The x-axis is exit thresholds, y-axis is entry thresholds, z-values are total_trades.

```typescript
const entryValues = [...new Set(zscoreData.results.map(r => r.entry))].sort((a, b) => a - b);
const exitValues = [...new Set(zscoreData.results.map(r => r.exit))].sort((a, b) => a - b);
const zMatrix = entryValues.map(entry =>
  exitValues.map(exit => {
    const row = zscoreData.results.find(r => r.entry === entry && r.exit === exit);
    return row?.total_trades ?? 0;
  })
);
```

Trace:
```typescript
{
  type: 'heatmap' as const,
  x: exitValues,
  y: entryValues,
  z: zMatrix,
  colorscale: 'Blues',
  colorbar: { title: { text: 'Trades' } },
}
```

Layout: `{ xaxis: { title: { text: 'Exit Threshold' } }, yaxis: { title: { text: 'Entry Threshold' } }, margin: { t: 48, b: 40, l: 56, r: 24 } }`

Height: 280px.
