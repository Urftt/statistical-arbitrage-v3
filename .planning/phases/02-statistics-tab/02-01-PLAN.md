**Data fetching (per D-06, D-08):**
- `useEffect` with cleanup flag pattern. Dependencies: `[asset1, asset2, timeframe, daysBack]`.
- Guard: `if (!asset1 || !asset2) return;`
- Sets loading true, error null before fetch.
- Calls `postCointegration({ asset1, asset2, timeframe, days_back: Number(daysBack) })`.
- On success: `setData(res)`. On error: `setError(err instanceof Error ? err.message : 'Failed to load statistics')`.
- `finally`: `if (!cancelled) setLoading(false)`.
- Cleanup: `return () => { cancelled = true; };`

**Badge helper functions (above the component, below constants):**
Define 5 functions that return `{ color: string; label: string }`:

1. `pValueBadge(v: number)`: `< 0.01` = green "Strong", `<= 0.05` = yellow "Moderate", else red "Weak"
2. `halfLifeBadge(v: number | null)`: `null` = gray "N/A", `< 20` = green "Fast", `<= 50` = yellow "Medium", else red "Slow"
3. `hedgeRatioBadge(v: number)`: `>= 0.5 && <= 2.0` = green "Balanced", else yellow "Skewed"
4. `correlationBadge(v: number)`: `> 0.7` = green "High", `>= 0.4` = yellow "Moderate", else red "Low"
5. `cointScoreBadge(v: number)`: `> 70` = green "Strong", `>= 40` = yellow "Moderate", else red "Weak"

**Threshold shapes function (above the component):**
```typescript
function buildZScoreShapes(entry: number, exit: number) {
  return [
    { type: 'line' as const, x0: 0, x1: 1, xref: 'paper' as const, y0: entry, y1: entry, line: { color: '#FF6B6B', width: 1, dash: 'dash' as const } },
    { type: 'line' as const, x0: 0, x1: 1, xref: 'paper' as const, y0: -entry, y1: -entry, line: { color: '#FF6B6B', width: 1, dash: 'dash' as const } },
    { type: 'line' as const, x0: 0, x1: 1, xref: 'paper' as const, y0: exit, y1: exit, line: { color: '#FCC419', width: 1, dash: 'dot' as const } },
    { type: 'line' as const, x0: 0, x1: 1, xref: 'paper' as const, y0: -exit, y1: -exit, line: { color: '#FCC419', width: 1, dash: 'dot' as const } },
  ];
}
```

**Render structure** (per UI-SPEC Layout Specification):

1. **Error state** — If `error` is set and `!loading`, render `Alert color="red" variant="light" icon={<IconAlertCircle size={16} />} title="Could not load statistics"` with body: `Failed to fetch cointegration data for ${asset1} / ${asset2}. Check that data is cached for this pair, then try again.` Then return early.

2. **Main layout** — `Stack gap="lg"`:

   a. **Lookback row** — `Group justify="space-between" align="center"`:
      - `Text size="sm" c="dimmed"` with text "Lookback period"
      - `Select size="sm" w={160} data={LOOKBACK_OPTIONS} value={daysBack} onChange={(v) => v && setDaysBack(v)}`

   b. **Stat cards** — If `loading`: `SimpleGrid cols={{ base: 2, sm: 3, lg: 5 }} spacing="md"` with 5x `Skeleton height={80}`.
      If `data`: same `SimpleGrid` with 5 `Paper p="md" radius="sm"` cards. Each card is a `Stack gap="xs"`:
      - `Text size="xs" c="dimmed" tt="uppercase"` — label (per Copywriting Contract: "P-Value", "Half-Life", "Hedge Ratio", "Correlation", "Coint. Score")
      - `Text size="lg" fw={700}` — formatted value:
        - P-value: `data.p_value.toFixed(4)` (e.g. "0.0031")
        - Half-life: `data.half_life !== null ? data.half_life.toFixed(1) + ' bars' : 'N/A'`
        - Hedge ratio: `data.hedge_ratio.toFixed(4)` (e.g. "0.9823")
        - Correlation: `data.correlation.toFixed(2)` (e.g. "0.87")
        - Coint. score: `Math.round(data.cointegration_score) + '/100'` (e.g. "78/100")
      - `Badge size="sm" variant="light" color={badge.color}` — text from badge helper

   c. **Spread chart** — If `loading`: `Skeleton height={260}`.
      If `data`: `Paper p="md"` wrapping `PlotlyChart` with:
      - `data={[{ type: 'scatter', x: dates, y: data.spread, mode: 'lines', name: 'Spread' }]}`
      - `layout={{ title: 'Spread', height: 260 }}`
      - `config={{ displayModeBar: 'hover' as const, scrollZoom: true }}`
      - Where `dates = data.timestamps.map((ts) => new Date(ts).toISOString())`

   d. **Z-score chart** — If `loading`: `Skeleton height={240}`.
      If `data`: `Paper p="md"` wrapping `PlotlyChart` with:
      - `data={[{ type: 'scatter', x: dates, y: data.zscore, mode: 'lines', name: 'Z-Score' }]}`
      - `layout={{ title: 'Z-Score', height: 240, shapes: buildZScoreShapes(entryThreshold, exitThreshold) }}`
      - `config={{ displayModeBar: 'hover' as const, scrollZoom: true }}`

   e. **Threshold sliders** — `Stack gap="sm"`:
      - Entry threshold: `Group justify="space-between"` with `Text size="sm"` "Entry Threshold" and `Text size="sm" fw={700}` showing `± ${entryThreshold.toFixed(1)}`. Below: `Slider min={0.5} max={4.0} step={0.1} value={entryThreshold} onChange={setEntryThreshold} color="red"`.
      - Exit threshold: Same pattern with "Exit Threshold", `± ${exitThreshold.toFixed(1)}`, `Slider min={0.0} max={2.0} step={0.1} value={exitThreshold} onChange={setExitThreshold} color="yellow"`.

**TypeScript strict mode compliance:**
- No `any` types. All state properly typed.
- `displayModeBar: 'hover' as const` to satisfy type checker. If this causes a type error, fall back to `displayModeBar: false`.
- Use `@/` path alias for all imports.

- Use Plotly subplot API — use two separate `PlotlyChart` instances.
