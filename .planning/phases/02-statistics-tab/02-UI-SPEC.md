## Spacing Scale

Mantine v8 spacing tokens map to the 8-point scale. Use Mantine prop values — never raw pixel values in component props.

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline badge padding |
| sm | 8px | Stack gap between compact elements, slider label gap |
| md | 16px | Default Stack gap, `pt="md"` on Tabs.Panel, card internal padding |
| lg | 24px | Section separation — gap between stat cards row and chart stack |
| xl | 32px | Major layout gap — gap between chart stack and page bottom |
| 2xl | 48px | Not used in Phase 2 |
| 3xl | 64px | Not used in Phase 2 |

Exceptions:
- Chart heights: spread chart 260px, z-score chart 240px (Plotly layout, not spacing tokens)
- Slider component occupies full width of its container (`w="100%"`)

Source: Phase 1 UI-SPEC (carried forward), CONTEXT.md layout decisions

## Color

Dark mode only. Mantine v8 dark color scheme. Never use light-mode variants.

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | `var(--mantine-color-dark-7)` — #1A1B1E | Page background, chart plot_bgcolor |
| Secondary (30%) | `var(--mantine-color-dark-6)` — #25262B | Paper cards (stat cards, chart wrappers) |
| Accent (10%) | `var(--mantine-color-blue-6)` — #228BE6 | Active tab pill (inherited from Phase 1), spread chart line color |
| Destructive | `var(--mantine-color-red-6)` — #FA5252 | "Weak" badge for concerning metrics (p-value > 0.05), error Alert |

Accent reserved for:
1. Active "Statistics" tab pill — exactly one tab active at a time (inherited from Phase 1, no change)
2. Spread chart line — the primary spread time-series trace
3. Active sidebar NavLink (inherited, no change)

Semantic badge colors (CONTEXT.md D-02):
- Green `var(--mantine-color-green-6)` — #51CF66: strong/good values ("Strong", "Fast", "High")
- Yellow `var(--mantine-color-yellow-6)` — #FAB005: moderate values ("Moderate", "Medium")
- Red `var(--mantine-color-red-6)` — #FA5252: weak/concerning values ("Weak", "Slow", "Low")

Badge color thresholds (Claude's discretion — filling gaps from CONTEXT.md):
- P-value: < 0.01 = green "Strong", 0.01–0.05 = yellow "Moderate", > 0.05 = red "Weak"
- Half-life: < 20 bars = green "Fast", 20–50 bars = yellow "Medium", > 50 bars = red "Slow"
- Hedge ratio: 0.5–2.0 = green "Balanced", < 0.5 or > 2.0 = yellow "Skewed"
- Correlation: > 0.7 = green "High", 0.4–0.7 = yellow "Moderate", < 0.4 = red "Low"
- Cointegration score: > 70 = green "Strong", 40–70 = yellow "Moderate", < 40 = red "Weak"

Chart line colors (from PLOTLY_DARK_TEMPLATE colorway — do not override):
- Spread trace: `#339AF0` (blue[5])
- Z-score trace: `#51CF66` (green[5])
- Entry threshold lines (+/−): `#FF6B6B` (red[5]), `dash="dash"`, `width=1`
- Exit threshold lines (+/−): `#FCC419` (yellow[5]), `dash="dot"`, `width=1`
- Zero line on z-score chart: `rgba(55, 58, 64, 0.8)` (matches gridcolor)

Source: `frontend/src/lib/theme.ts` (PLOTLY_DARK_TEMPLATE colorway), CONTEXT.md D-02, Phase 1 UI-SPEC

## Interaction Contracts

### Auto-Load on Tab Open
- When the Statistics tab is active and `asset1` + `asset2` are both non-empty, `useEffect` fires `postCointegration({ asset1, asset2, timeframe, days_back })` immediately.
- Shows `Skeleton` for stat cards (5 × `Skeleton height={80}`) and chart areas (`Skeleton height={260}` + `Skeleton height={240}`) during fetch.
- On success: replaces skeletons with stat cards and charts.
- On error: replaces all content with a single `Alert` (see Copywriting Contract).
- Cleanup flag pattern: `let cancelled = false` to prevent state update on unmounted component.

Source: CONTEXT.md D-06; CLAUDE.md useEffect pattern

### Lookback Dropdown
- `Select` with options: `[{ value: "90", label: "90 days" }, { value: "180", label: "180 days" }, { value: "365", label: "1 year" }, { value: "730", label: "2 years" }]`
- Default value: `"365"` (1 year) — Claude's discretion; CONTEXT.md D-07 leaves this open.
- Changing selection triggers immediate re-fetch (no separate button). Stat cards and charts re-enter skeleton state.
- `days_back` is passed as a number to `postCointegration()`.
- Placement: above the stat cards row, right-aligned in a `Group justify="space-between"` alongside a `Text size="sm" c="dimmed"` label "Lookback period".

Source: CONTEXT.md D-07, D-08

### Z-Score Threshold Sliders
- Two `Slider` components, each full-width, rendered in a `Stack gap="sm"` below the chart block.
- Entry threshold slider: range `0.5`–`4.0`, step `0.1`, default `2.0`. Draws horizontal lines at `+value` and `−value` on the z-score chart.
- Exit threshold slider: range `0.0`–`2.0`, step `0.1`, default `0.5`. Draws horizontal lines at `+value` and `−value` on the z-score chart.
- Slider `value` stored in local `useState`. Chart shapes array updated on every slider change (no re-fetch; Plotly `shapes` prop updated in-place).
- Each slider has a label row: `Group justify="space-between"` with label text left and current value right (`Text size="sm" fw={700}`).
- Sliders are always visible when statistics are loaded — not hidden behind a toggle.

Source: CONTEXT.md D-04, D-05; Claude's discretion for range/step/defaults

### Chart Interactions
- Plotly zoom: drag-select to zoom, scroll to zoom (Plotly default).
- Plotly pan: enabled via `config={{ scrollZoom: true }}`.
- Reset zoom: Plotly reset button (shown via `displayModeBar: true` only on hover — use `config={{ displayModeBar: 'hover' }}`).
- Hover tooltips: enabled on both charts, show date + value.
- Both charts share the same x-axis domain (same timestamps array). They are rendered as two separate `PlotlyChart` instances stacked vertically — not as Plotly subplots.

Source: CONTEXT.md D-10, D-11; `frontend/src/components/charts/PlotlyChart.tsx` (config defaults)

### Pair Change
- `StatisticsTab` receives `asset1`, `asset2`, `timeframe` from `PairContext`.
- The parent `Tabs` container in `page.tsx` already uses `key={`${asset1}-${asset2}`}` to remount on pair change. This causes `StatisticsTab` to remount, clearing all local state automatically.
- No explicit reset logic needed inside `StatisticsTab`.

Source: CONTEXT.md; `frontend/src/app/(dashboard)/pair-analysis/page.tsx` (key prop, line 64)

## Layout Specification

### Statistics Tab Structure

```
Tabs.Panel value="statistics" pt="md"
└── StatisticsTab
    └── Stack gap="lg"
        ├── Group justify="space-between" align="center"
        │   ├── Text size="sm" c="dimmed" — "Lookback period"
        │   └── Select — lookback dropdown (size="sm", w={160})
        │
        ├── [Loading] SimpleGrid cols={{ base: 2, sm: 3, lg: 5 }} spacing="md"
        │   └── Skeleton height={80} × 5
        │
        ├── [Loaded] SimpleGrid cols={{ base: 2, sm: 3, lg: 5 }} spacing="md"
        │   ├── Paper p="md" — P-Value card
        │   │   └── Stack gap="xs"
        │   │       ├── Text size="xs" c="dimmed" tt="uppercase" — "P-Value"
        │   │       ├── Text size="lg" fw={700} — e.g. "0.0031"
        │   │       └── Badge size="sm" variant="light" color={...} — "Strong"
        │   ├── Paper p="md" — Half-Life card
        │   ├── Paper p="md" — Hedge Ratio card
        │   ├── Paper p="md" — Correlation card
        │   └── Paper p="md" — Cointegration Score card
        │
        ├── [Loading] Skeleton height={260}
        ├── [Loaded] Paper p="md" — Spread chart wrapper
        │   └── PlotlyChart height={260} — spread time series
        │
        ├── [Loading] Skeleton height={240}
        ├── [Loaded] Paper p="md" — Z-score chart wrapper
        │   └── PlotlyChart height={240} — z-score time series with threshold shape lines
        │
        └── Stack gap="sm" — threshold slider controls
            ├── Group justify="space-between"
            │   ├── Text size="sm" — "Entry Threshold"
            │   └── Text size="sm" fw={700} — e.g. "± 2.0"

            └── Slider min={0.0} max={2.0} step={0.1} defaultValue={0.5}
```

### Error State (replaces entire Stack)

```
Tabs.Panel value="statistics" pt="md"
└── Alert color="red" variant="light" icon={<IconAlertCircle size={16} />}
    title="Could not load statistics"
    └── [error body text — see Copywriting Contract]
```

### Stat Card Format

Each of the 5 stat cards follows the identical internal layout:
- `Paper p="md" radius="sm"` — secondary background color applied via Mantine Paper default
- `Stack gap="xs"` inside
- Row 1: `Text size="xs" c="dimmed" tt="uppercase"` — metric name
- Row 2: `Text size="lg" fw={700}` — formatted value
- Row 3: `Badge size="sm" variant="light" color={semantic}` — interpretive label

Formatted values:
- P-value: 4 decimal places, e.g. "0.0031"
- Half-life: 1 decimal place + " bars", e.g. "14.2 bars"
- Hedge ratio: 4 decimal places, e.g. "0.9823"
- Correlation: 2 decimal places, e.g. "0.87"
- Cointegration score: integer + "/100", e.g. "78/100"

Source: CONTEXT.md D-01, D-02, D-03; Claude's discretion for value formatting
