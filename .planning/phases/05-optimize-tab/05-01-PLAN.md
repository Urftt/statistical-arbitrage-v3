1. `buildAxisValues(axis: ParameterAxisPayload): number[]` -- loop from min to max+step/2, step by step, push `Math.round(v * 1e10) / 1e10`. This matches Python's `round(float(v), 10)` rounding.

2. `getMetricValue(cell: GridSearchCellPayload, metric: string): number | null` -- switch on metric: `'sharpe_ratio'` returns `cell.metrics.sharpe_ratio`, `'total_pnl'` returns `cell.metrics.total_net_pnl`, `'win_rate'` returns `cell.metrics.win_rate`, `'max_drawdown'` returns `cell.metrics.max_drawdown_pct`, default returns `null`.

3. `colorscaleForMetric(metric: string): string` -- `'win_rate'` returns `'Blues'`, `'max_drawdown'` returns `'RdYlGn'` (NOT reversed -- lower drawdown is better, so green=low, red=high is correct for RdYlGn where low values map to red. Actually, we want lower drawdown = green, so use `'RdYlGn_r'` to reverse it since drawdown numbers are positive and lower is better). For all others return `'RdYlGn'`.
   CORRECTION per UI-SPEC: `RdYlGn` for Sharpe/P&L (higher = better = green), `Blues` for win rate, reversed `RdYlGn` for max drawdown (lower = better).

4. `robustnessLabel(score: number | null): { label: string; color: string }` -- per UI-SPEC: score >= 0.7 returns `{ label: 'Strong', color: 'green' }`, score >= 0.4 returns `{ label: 'Moderate', color: 'yellow' }`, else returns `{ label: 'Weak', color: 'red' }`. async function handleGridSearch() {
  cancelRef.current = false;
  setGridLoading(true);
  setGridError(null);
  try {
    const req: GridSearchRequest = {
      asset1: `${asset1}/EUR`,
      asset2: `${asset2}/EUR`,
      timeframe,
      days_back: 365,
      axes: [
        { name: axis1Param, min_value: axis1Min, max_value: axis1Max, step: axis1Step },
        { name: axis2Param, min_value: axis2Min, max_value: axis2Max, step: axis2Step },
      ],
      base_strategy: { ...baseStrategy, min_trade_count_warning: 3 },
      optimize_metric: 'sharpe_ratio',
      max_combinations: 500,
    };
    const res = await postGridSearch(req);
    if (!cancelRef.current) setGridData(res);
  } catch (err: unknown) {
    if (!cancelRef.current) setGridError(err instanceof Error ? err.message : 'Unknown error');
  } finally {
    if (!cancelRef.current) setGridLoading(false);
  }
}
